## About

<a id="about"></a>

I build backend-heavy full stack products — authentication systems, relational and document data models, caching layers, and cloud-deployed infrastructure — under the solo freelance practice **Vixora**. I run a multi-agent AI workflow (Antigravity/Gemini for execution across parallel agents, this Claude instance for UI systems and design), with final architectural and quality review always sitting with me.

Currently a **B.Tech Computer Science & Engineering** student at Nirma University (2023–2027, CGPA 8.29), shipping production systems alongside coursework rather than treating them as class projects.
